Condiciones de Dirichlet : Resolución en diferencias finitas¶

$$ u''(x) = f(x)$$$$u(0)=u(L)=0$$

$$f(x) = 1$$

Método¶

Aproximacin de la segunda derivada, dado $h = \frac{L}{N}$ $$u''(x_i) = \frac{u_{i+1}-2u_i+u_{i-1}}{h^2} = f_i = f(x_i)$$

def solve1(f, Np, ini=0.0, fin=1.0):
  N=Np
  a=ini
  L=fin

  h = (L-a)/N
  nodos = [a+h*i for i in np.arange(N+1)]
  F = np.array([f(x) for x in nodos[1:-1]])
  M = np.zeros((N-1, N-1), dtype=float)

  for i in range(N-1):
    M[i, i] = -2.0
    if i<N-2:
      M[i, i+1]=1.0
      M[i+1, i]=1.0

  U = h**2*np.linalg.inv(M)@F
  U = np.hstack(([0.0], U, [0.0]))

  return nodos, U

def solve2(f, Np, ini=0.0, fin=1.0):
  N=Np
  a=ini
  L=fin


  h = (L-a)/N
  nodos = [a+h*i for i in np.arange(N+1)]
  F = np.array([f(x) for x in nodos[1:-1]])
  diagonals = [-2*np.ones(N-1), np.ones(N-2), np.ones(N-2)]
  M = diags(diagonals, (0, 1, -1)).toarray()

  U = h**2*np.linalg.inv(M)@F
  U = np.hstack(([0.0], U, [0.0]))

  return nodos, U

def solve3(f, Np, ini=0.0, fin=1.0, ua=0, ub=0):
  N=Np
  a=ini
  L=fin


  h = (L-a)/N
  nodos = [a+h*i for i in np.arange(N+1)]
  F = np.array([f(x) for x in nodos])
  F[0]=ua
  F[N]=ub
  diagonals = [-2*np.ones(N+1), np.ones(N), np.ones(N)]
  M = diags(diagonals, (0, 1, -1)).toarray()
  M[0, 0]=h**2
  M[0, 1]=0
  M[N, N-1]=0
  M[N, N]=h**2

  U = h**2*np.linalg.inv(M)@F

  return nodos, U

Segundo ejemplo : Resolución en diferencias finitas¶

$$ u''(x) + u(x) = f(x)$$$$u(0)=u(L)=0$$

$$f(x) = 0$$

Método¶

Aproximacin de la segunda derivada, dado $h = \frac{L}{N}$ $$u''(x_i) + u(x_i) = \frac{u_{i+1}-2u_i+u_{i-1}}{h^2} + u_i = f_i = f(x_i)$$

def solve4(f, Np, ini=0.0, fin=1.0, ua=0, ub=0):
  N=Np
  a=ini
  L=fin


  h = (L-a)/N
  nodos = [a+h*i for i in np.arange(N+1)]
  F = np.array([f(x) for x in nodos])
  F[0]=ua
  F[N]=ub
  diagonals = [-2*np.ones(N+1), np.ones(N), np.ones(N)]
  M = diags(diagonals, (0, 1, -1)).toarray()+(np.identity(N+1)*h**2)
  M[0, 0]=h**2
  M[0, 1]=0
  M[N, N-1]=0
  M[N, N]=h**2


  U = h**2*np.linalg.inv(M)@F

  return nodos, U

Condiciones de tipo Neumann:¶

$$ u''(x) = f(x) \textrm{ en } [0,1]$$$$u'(0) = 0$$

y $$u(1)=0$$

Método¶

El mismo que en el primer problema, pero debemos introducir una discretizacin dedicada a implementar las condiciones de la derivada, se hace calculando los coeficientes imponiendo exactitud, se obtienen los siguientes:

$$ u'(0) = 0 \Longrightarrow \frac{-\frac{3}{2} u_0 + 2u_1 - \frac{1}{2}u_2}{h} $$

Vamos a hacer una implementacin lo ms general posible, teniendo en cuenta que si ambas condiciones se colocan en la derivada, tenemos infinitas soluciones y el problema no est bien definido.

def solve5(f, Np, ini=0.0, fin=1.0, ua=0, ub=0, n1=False, n2=False):
  if (n1 and n2):
    print("EL PROBLEMA NO EST BIEN DEFINIDO")
  N=Np
  a=ini
  L=fin


  h = (L-a)/N
  nodos = [a+h*i for i in np.arange(N+1)]
  F = np.array([f(x) for x in nodos])
  F[0]=ua
  F[N]=ub
  diagonals = [-2*np.ones(N+1), np.ones(N), np.ones(N)]
  M = diags(diagonals, (0, 1, -1)).toarray()

  M[0, 0]=h**2
  M[0, 1]=0
  M[N, N-1]=0
  M[N, N]=h**2

  if(n1):
    M[0, 0]=-3*h
    M[0, 1]=4*h
    M[0, 2]=-1*h

  if(n2):
    M[N, N]=-3*h
    M[N, N-1]=4*h
    M[N, N-2]=-1*h


  U = h**2*np.linalg.inv(M)@F

  return nodos, U
